# 1A2B 猜數字遊戲 (8086 組合語言)

這是一個使用 8086 組合語言 (TASM/MASM) 撰寫的經典 1A2B 猜數字遊戲。

## 📅 功能特色

* **真隨機數種子**: 使用 `INT 1Ah` 讀取系統時鐘，初始化隨機數種子 (`randSeed`)，確保每次執行的答案都不同。
* **答案不重複**: `GenerateAnswer` 程序使用一個 10 位元的 `table` (雜湊表) 來確保產生的 4 位數答案中，數字皆不重複。
* **即時鍵盤輸入**: 透過 `INT 16h` 接收鍵盤輸入，並使用 `INT 10h` (Teletype) 模式即時回顯。
* **完整遊戲邏輯**:
    * 自動判斷玩家是否輸入了 4 位數字。
    * `CheckAnswer` 程序使用雙重迴圈，正確計算 `A` (位置與數字皆對) 和 `B` (僅數字對)。
* **彩色介面**:
    * 使用 `PRINT_COLOR` 巨集顯示彩色的 A/B 結果 (A 顯示為綠色，B 顯示為紅色)。
    * 按 F1 顯示答案時，答案以黃色顯示。
* **便利的控制項**:
    * **ESC**: 立即結束遊戲。
    * **F1**: 顯示正確答案並結束遊戲。

## 🗂️ 檔案結構

* `game.asm`
    * 遊戲的主程式碼。
    * 包含所有遊戲邏輯、子程序 (Procedures)，並在開頭 `include` 了 `print.h`。
* `print.h`
    * 一個巨集 (Macro) 庫。
    * 提供了 `CLS` (清螢幕)、`PRINTSTR` (印字串)、`NEWLINE` (換行) 和 `PRINT_COLOR` (彩色印字) 。
